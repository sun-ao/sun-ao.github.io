Hugo + Github Pages 搭建 Blog

Written by sun.ao with ♥ on March 16, 2021 in 105 words

安装hugo

# windows
https://github.com/gohugoio/hugo/releases
# macOS
brew install hugo

初始化

# hugo-blog 为自定义文件夹
hugo new site hugo-blog
cd hugo-blog
# 下载主题 （根据需求自定）
git init
git submodule add https://github.com/sun-ao/mogege themes/mogege
# 模版演示
cp -R themes/mogege/exampleSite/content .
cp themes/mogege/exampleSite/config.toml .
# 调试
hugo server

发布

hugo
cd public
git init
# sun-ao 填写自己的 github 用户名
git remote add origin https://github.com/sun-ao/sun-ao.github.io.git
git add .
git commit -m 'post'
git push -f --set-upstream origin master
