Windows Subsystem for Linux

Written by sun.ao with ♥ on March 17, 2021 in 266 words

开启

# 管理员身份运行 PowerShell
Enable-WindowsOptionalFeature -Online -FeatureName Microsoft-Windows-Subsystem-Linux

选择Linux发行版本

  • Ubuntu （打开 Microsoft Store，搜索「WSL」）
# 查询Ubuntu版本 
cat /etc/issue

# Ubuntu 20.04 LTS \n \l
  • 软件镜像源
cd /etc/apt

# 备份
sudo cp sources.list sources.list.20200731

# 更新
find -name 'sources.list' | sudo xargs perl -pi -e 's|http://archive.ubuntu.com/ubuntu/|https://mirrors.tuna.tsinghua.edu.cn/ubuntu/|g' 

find -name 'sources.list' | sudo xargs perl -pi -e 's|http://security.ubuntu.com/ubuntu/|https://mirrors.tuna.tsinghua.edu.cn/ubuntu/|g' 

# 更新源，使配置生效
sudo apt update

WSL 与 Windows 文件互通

  • WSL：/mnt
  • Windows：C:\Users\...\AppData\Local\Packages\...UbuntuonWindows_...\LocalState\rootfs

重置系统

在开始菜单对应的而应用图标点击右键，选择更多，点击应用设置，在新打开的窗口找到重置按钮，点击即可！
