早上八点，广州老城区。

茶楼里人声鼎沸，阿公阿婆们已经坐满了桌。服务员推着小车穿梭其间，车上摆满了蒸笼——虾饺、烧卖、叉烧包、凤爪……阿婆熟练地翻着报纸，阿公慢悠悠地品着普洱茶，一盅两件，可以从早吃到中午。

这就是广东人的早茶，一种刻在骨子里的生活方式。

广东，是一个让人来了就想吃的城市。

省份名片
#

  • 地理位置：中国南部沿海，南岭以南
  • 最佳旅行季节：10月-次年4月（避开湿热夏季）
  • 特色标签：早茶文化、岭南风情、美食天堂、改革开放

必去景点
#

广州：千年商都的烟火气
#

广州是一座有2000多年历史的城市，也是中国最包容的城市之一。

陈家祠是岭南建筑的瑰宝，三进院落，装饰着精美的木雕、石雕、砖雕、陶塑。每一处细节，都是工匠的心血。

沙面是珠江边的小岛，曾是英法租界。岛上遍布欧式老建筑，榕树参天，是广州最浪漫的地方。

广州塔是城市的新地标，600米高，有"小蛮腰"之称。晚上灯光亮起，是广州最美的夜景。

最佳体验：早上喝早茶，下午逛沙面，晚上登广州塔看夜景。

深圳：从小渔村到创新之都
#

深圳是中国最年轻的一线城市，40年前还是一个小渔村。

世界之窗把世界各地的名胜微缩在一起，一天"环游世界"。华侨城创意园是文艺青年的聚集地，老厂房变成了画廊、咖啡馆、设计工作室。

大鹏所城是深圳的历史记忆，600多年的古城，青砖黛瓦，诉说着这座城市的过去。

最佳体验：去华强北逛电子市场，感受"中国硅谷"的活力。

潮汕：美食的朝圣之地
#

潮汕地区（潮州、汕头、揭阳）是美食家的天堂。

潮州古城有千年历史，牌坊街两旁是骑楼老建筑，小吃店一家挨着一家。牛肉丸、蚝烙、粿条、鸭母捻……每一样都让人垂涎。

广济桥是中国四大古桥之一，中间由18艘木船连成浮桥，每天早晚拆桥、搭桥，是"世界上最早的启闭式桥梁"。

最佳体验：在潮州住几天，把每条巷子里的小吃都吃一遍。

特色体验
#

顺德寻味
#

顺德是"世界美食之都"，粤菜的发源地之一。

顺德鱼生是当地名菜，鱼片薄如蝉翼，蘸着姜葱蒜油，鲜甜爽滑。双皮奶是顺德甜品，奶香浓郁，入口即化。均安蒸猪、伦教糕、大良炒牛奶……每一道都是经典。

开平碉楼
#

开平碉楼是世界文化遗产，中西合璧的奇特建筑。

百年前，华侨们把在海外看到的建筑风格带回家乡，建起了一座座碉楼。古希腊的柱廊、古罗马的拱券、伊斯兰的穹顶……各种风格混搭，却意外地和谐。

丹霞山
#

丹霞山是世界自然遗产，红色的砂岩峰林，像燃烧的火焰。长老峰、阳元石、翔龙湖，每一处都是大自然的鬼斧神工。

旅行小贴士
#

  • 交通：广州、深圳是交通枢纽，高铁、飞机直达全国各地。潮汕也有高铁站。
  • 住宿：广州建议住老城区（荔湾、越秀）体验早茶文化；深圳建议住福田或南山。
  • 注意：广东夏季湿热，建议秋冬来访。早茶要早去，晚了要排队。

广东的魅力，藏在那些烟火气里。

早茶楼里阿公阿婆的闲聊，夜市大排档的喧嚣，潮汕巷子里飘出的牛肉香……这些日常，才是广东最动人的风景。

来广东，用舌尖丈量这片热土。

明天，我们去广西，看看桂林的山水、阳朔的田园。
