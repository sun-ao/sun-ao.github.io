秋天，九寨沟。

湖水呈现出梦幻般的蓝绿色，像一块块宝石镶嵌在山谷中。五花海的色彩斑斓，珍珠滩的瀑布飞泻，长海的深邃宁静……每一处都美得让人窒息。站在湖边，看倒映在水中的彩林，分不清哪是天，哪是水。

“九寨归来不看水”，此言不虚。

四川，是一个让人来了就不想走的地方。

省份名片
#

  • 地理位置：中国西南部，长江上游
  • 最佳旅行季节：9-11月（九寨沟秋色）、全年（成都）
  • 特色标签：九寨仙境、熊猫故乡、川菜美食、天府之国

必去景点
#

九寨沟：人间仙境
#

九寨沟是世界自然遗产，“人间仙境”。

五花海是九寨沟最美的地方，湖水呈现出蓝、绿、黄等多种颜色，像一块调色板。诺日朗瀑布是中国最宽的高原瀑布，宽270米，气势磅礴。长海是九寨沟最大的海子，深邃宁静。珍珠滩瀑布是《西游记》的取景地，水流在钙华滩上跳跃，像无数珍珠。

最佳体验：秋天来九寨沟，看彩林倒映在湖水中。住在沟口，早上第一批进沟，避开人流。

成都：一座来了就不想走的城市
#

成都是中国最悠闲的城市，“少不入川，老不出蜀”。

宽窄巷子是成都的名片，青砖灰瓦的老街，茶馆、小吃、文创店林立。找一家茶馆，点一杯盖碗茶，掏一次耳朵，发一下午呆。

锦里是成都最热闹的古街，小吃、手工艺品、川剧表演，应有尽有。春熙路是成都最繁华的商业街，美女如云，时尚潮流。

大熊猫繁育研究基地是看大熊猫的最佳地点，早上来，看熊猫吃竹子、打滚、睡觉，萌化你的心。

最佳体验：在茶馆里坐一下午，看成都人的悠闲生活。晚上吃火锅，感受成都的火辣。

峨眉山：佛教名山
#

峨眉山是中国四大佛教名山之一，普贤菩萨的道场。

金顶是峨眉山的最高点，海拔3079米。清晨，看日出、云海、佛光，是峨眉山三大奇观。万年寺是峨眉山最古老的寺庙，无梁砖殿是建筑奇迹。

最佳体验：在山上住一晚，清晨看日出。下山时走山路，看沿途的猴群。

乐山大佛：世界最大石刻佛像
#

乐山大佛是世界上最大的石刻佛像，高71米，开凿于唐代。

大佛依山而建，面容慈祥，俯瞰三江汇流。可以坐船远观大佛全貌，也可以走栈道近距离观看。

特色体验
#

稻城亚丁：蓝色星球上的最后一片净土
#

稻城亚丁被誉为"蓝色星球上的最后一片净土"。

三神山——仙乃日、央迈勇、夏诺多吉，终年积雪，神圣庄严。牛奶海、五色海是高原湖泊，湖水清澈，倒映着雪山。洛绒牛场是徒步的起点，草原、溪流、雪山，风景如画。

最佳体验：秋天来亚丁，看金黄的草甸和洁白的雪山。徒步转山，感受藏区的神圣。

都江堰与青城山
#

都江堰是战国时期李冰父子修建的水利工程，两千多年仍在使用，是世界水利文化的鼻祖。

青城山是道教名山，“青城天下幽”。山间古树参天，道观林立，是避暑的好去处。

川菜美食
#

四川是美食天堂。火锅是四川的名片，红油翻滚，麻辣鲜香。麻婆豆腐、回锅肉、宫保鸡丁是川菜经典。串串香、冒菜、钵钵鸡是街头小吃，便宜又好吃。

成都小吃更是丰富，龙抄手、钟水饺、担担面、甜水面、三大炮……每一样都值得尝试。

旅行小贴士
#

  • 交通：成都是四川的交通枢纽，双机场（双流、天府），高铁直达全国各地。九寨沟有机场，也可以从成都坐大巴。
  • 住宿：成都建议住春熙路或宽窄巷子附近；九寨沟建议住沟口；峨眉山建议住山上或山脚。
  • 注意：九寨沟海拔高，注意高原反应。川菜麻辣，量力而行。看熊猫要早去，熊猫上午活跃。

四川的魅力，在于它的悠闲。

在成都的茶馆里发呆，在九寨沟的湖边漫步，在大熊猫基地看萌宠，在火锅店里大快朵颐……时间在这里，好像过得特别慢。

来四川，学会慢下来。

明天，我们去贵州，看看黄果树瀑布的壮观、苗寨的风情。
