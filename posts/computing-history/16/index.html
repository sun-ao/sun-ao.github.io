1995年8月，加州山景城。

一家成立仅16个月的公司正在准备上市。

公司只有50名员工，产品是免费的浏览器，从未盈利。

但投资者疯狂追捧。上市当天，股价从28美元涨到75美元。公司市值达到29亿美元。

这家公司叫网景。它的产品叫Navigator。

它的创始人马克·安德森，只有24岁。

这是互联网泡沫的开始，也是浏览器战争的序幕。

网景的崛起
#

故事要从1992年说起。

伊利诺伊大学的本科生马克·安德森，在学校的国家超级计算应用中心（NCSA）打工。他接触到了万维网，但当时的浏览器体验很差——只能显示文字，操作复杂。

安德森想：能不能做一个更好的浏览器？

他和同事埃里克·比纳一起，开发了Mosaic浏览器。

Mosaic的特点是：

  • 图形界面：可以显示图片，不只是文字
  • 鼠标操作：点击链接就能跳转
  • 跨平台：可以在Windows、Mac、Unix上运行
  • 免费下载：任何人都可以使用

Mosaic发布后，迅速流行。一年内被下载了数百万次。

但安德森和大学产生了分歧。大学拥有Mosaic的版权，安德森无法商业化。他决定离开，自己创业。

网景通信公司
#

1994年，安德森遇到了硅谷传奇人物吉姆·克拉克。

克拉克是硅谷的连续创业者，创办了Silicon Graphics（SGI）。他刚离开SGI，正在寻找新机会。

安德森向他展示了万维网的潜力。克拉克被说服了，投资400万美元，创办了网景通信公司。

安德森招募了Mosaic的核心开发团队，重新开发浏览器。为了避免版权纠纷，他们从头写起，不使用任何Mosaic代码。

1994年12月，Netscape Navigator 1.0发布。

它比Mosaic更快、更稳定、功能更强。而且，它仍然是免费的——个人用户可以免费下载，企业用户需要付费。

Navigator迅速占领市场。到1995年，它占据了90%的浏览器市场份额。

网景成为互联网时代的第一个明星公司。

微软的觉醒
#

在雷德蒙德，微软总部，比尔·盖茨注意到了网景。

盖茨最初低估了互联网。他在1994年的备忘录中写道：“互联网只是暂时的热潮。”

但网景的成功让他警觉。网景不仅控制了浏览器，还在开发服务器软件、电子邮件、群件——这些都是微软的地盘。

更危险的是：浏览器可能成为新的平台。

如果用户通过浏览器访问应用，操作系统就变得不那么重要了。Windows可能被边缘化。

盖茨在1995年5月的备忘录中写道：“互联网是自PC以来最重要的发展。我们必须全力以赴。”

IE的反击
#

微软迅速行动。

他们从Spyglass公司购买了浏览器技术，改名为Internet Explorer。

1995年8月，IE 1.0发布，作为Windows 95的附加组件。

但IE 1.0很粗糙，无法与Navigator竞争。

真正的反击从IE 3.0开始。1996年8月，IE 3.0发布，功能与Navigator 3.0相当。更重要的是，IE是免费的，而且与Windows捆绑。

用户买Windows，就免费得到IE。为什么要下载Navigator？

网景的商业模式被击穿了。

捆绑策略
#

微软的捆绑策略极其有效：

第一，IE免费。 网景对个人用户免费，但对企业用户收费。IE对所有用户免费。

第二，IE预装。 Windows用户不需要下载，IE已经在电脑上了。

第三，IE不可卸载。 IE与Windows深度集成，用户无法完全移除。

第四，IE独占。 微软与电脑制造商签订协议，要求他们在Windows桌面上只显示IE图标。

到1997年，IE的市场份额从0增长到30%。网景的市场份额从90%下降到60%。

网景的挣扎
#

网景试图反击。

他们开发了Communicator——一个包含浏览器、电子邮件、新闻阅读器的套件。但软件臃肿，性能下降。

他们试图把浏览器做成平台，开发了JavaScript和插件技术。但这些技术还不够成熟。

他们向政府投诉微软垄断行为。

1997年，网景裁员，股价下跌。公司陷入困境。

反垄断诉讼
#

1998年5月，美国司法部和20个州对微软提起反垄断诉讼。

指控包括：

  • 利用Windows垄断地位，强制捆绑IE
  • 与电脑制造商签订排他性协议
  • 故意使Windows与网景软件不兼容

审判持续了两年。关键证据是微软内部的邮件，显示盖茨明确要求"切断网景的空气供应"。

2000年，法院判决微软违反反垄断法，应该拆分为两家公司：一家经营操作系统，一家经营应用软件。

微软上诉。2001年，上诉法院推翻了拆分判决，但确认微软违反了反垄断法。

最终，微软与政府达成和解：微软必须公开Windows的部分技术信息，不能对电脑制造商进行报复。

网景的结局
#

1998年11月，网景被**美国在线（AOL）**以42亿美元收购。

这是互联网时代第一个大型收购案。但AOL不懂浏览器业务，网景逐渐衰落。

2003年，网景解散。剩下的开发人员成立了Mozilla基金会，继续开发开源浏览器。

2008年，网景正式停止支持。

网景死了，但它的遗产永存：

  • JavaScript：网景开发的脚本语言，成为Web开发的标准
  • SSL/HTTPS：网景开发的安全协议，保护互联网通信
  • Mozilla/Firefox：网景的开源项目，后来发展成Firefox浏览器

Firefox的复兴
#

2004年，Mozilla基金会发布Firefox 1.0。

Firefox继承了网景的遗产，但更轻量、更安全、更可定制。它支持标签页浏览、扩展插件、隐私保护。

Firefox从IE手中夺回市场份额。到2010年，Firefox占据约25%的市场。

更重要的是，Firefox证明了：开源软件可以与商业软件竞争。

Chrome的崛起
#

2008年，Google发布Chrome浏览器。

Chrome的特点是：

  • 极速：使用V8引擎，JavaScript执行速度远超竞争对手
  • 简洁：界面极简，把空间留给网页内容
  • 稳定：每个标签页独立进程，一个崩溃不影响其他
  • 开源：基于Chromium开源项目

Chrome迅速崛起。到2012年，Chrome超越IE，成为市场份额第一的浏览器。

到2024年，Chrome占据约65%的市场份额。Safari约18%，Edge约5%，Firefox约3%。

浏览器的今天
#

今天的浏览器已经不仅是网页查看器，而是应用平台：

  • Web应用：Gmail、Google Docs、Figma都在浏览器中运行
  • 扩展生态：Chrome商店有数十万扩展
  • 开发者工具：浏览器内置调试工具
  • PWA：渐进式Web应用，可以像原生应用一样安装

浏览器战争从未真正结束。Chrome、Safari、Edge、Firefox仍在竞争。新的竞争者如Brave、Arc也在挑战现状。

下一步：编程语言
#

浏览器需要编程语言来创建交互式网页。网景开发的JavaScript，成为Web开发的标准。

但编程语言的历史远比JavaScript更悠久。从机器码到汇编，从C到Python，编程语言的进化反映了人类与计算机交互方式的演变。

明天，我们来讲编程语言的进化史。

今日知识点
#

浏览器战争（Browser Wars） 1990年代网景Navigator与微软IE的竞争。微软凭借Windows垄断地位，免费捆绑IE，最终击败网景。这场竞争引发了微软的反垄断诉讼，也推动了Web技术的发展。

捆绑销售（Bundling） 把多个产品打包销售。微软把IE与Windows捆绑，免费提供浏览器。这击穿了网景的商业模式，但也引发了反垄断诉讼。

JavaScript 网景开发的脚本语言，用于在网页中添加交互功能。JavaScript后来成为Web开发的标准，现在是世界上最流行的编程语言之一。

思考题
#

  1. 微软凭借Windows垄断地位击败了网景。你觉得这种竞争方式公平吗？
  2. 网景虽然失败了，但它开发的JavaScript、SSL等技术至今仍在使用。你觉得一个公司的"遗产"比"成功"更重要吗？

明天预告：编程语言进化论——从机器码到Python，人类如何让计算机听懂自己的语言？
