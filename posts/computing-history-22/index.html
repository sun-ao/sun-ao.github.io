2005年7月，美国加州。

Google悄悄收购了一家小公司，花了5000万美元。

这家公司叫Android Inc.，只有十几名员工。他们在开发一个手机操作系统。

创始人安迪·鲁宾曾经是苹果的工程师，后来创办了Danger公司（开发了T-Mobile Sidekick手机）。他相信：智能手机的未来是开放的。

Google看到了移动互联网的机会。他们不想只做搜索，他们想在手机上占据一席之地。

收购Android，是Google进入移动领域的第一步。

Android的诞生
#

Android团队加入Google后，继续开发操作系统。

他们的目标是：一个开放、免费、可定制的手机操作系统。

与iOS不同，Android：

  • 开源：源代码公开，厂商可以修改
  • 免费：厂商不需要支付授权费
  • 可定制：厂商可以添加自己的界面和应用
  • 多硬件支持：支持各种配置的手机

这让Android可以运行在各种手机上——从几百元的入门机到几千元的旗舰机。

第一款Android手机
#

2008年10月，HTC Dream（也叫T-Mobile G1）发布，这是第一款Android手机。

它有一个3.2英寸触摸屏，但也有一个滑出式物理键盘。这反映了当时的保守——很多人还不信任虚拟键盘。

G1运行Android 1.0，功能有限：

  • 通知栏、桌面小部件
  • Android Market（后来改名为Google Play）
  • Gmail、Google Maps、YouTube集成

G1销量一般，但它证明了Android可以工作。

Android的进化
#

Android快速进化，每年发布一个新版本：

Android 1.5 Cupcake（2009）：虚拟键盘、视频录制

Android 2.0 Eclair（2009）：多账户支持、Google Maps导航

Android 2.2 Froyo（2010）：移动热点、速度提升

Android 2.3 Gingerbread（2010）：NFC支持、改进的游戏支持

Android 4.0 Ice Cream Sandwich（2011）：全新的界面设计，统一手机和平板

Android 4.1 Jelly Bean（2012）：Google Now语音助手

Android 5.0 Lollipop（2014）：Material Design设计语言

Android 6.0 Marshmallow（2015）：权限管理、指纹识别

Android 7.0 Nougat（2016）：分屏多任务

Android 8.0 Oreo（2017）：通知渠道、画中画

Android 9.0 Pie（2018）：手势导航

Android 10（2019）：暗色主题、完全手势导航

Android 11（2020）：聊天气泡、一次性权限

Android 12（2021）：Material You设计语言

Android 13（2022）：隐私增强

Android 14（2023）：性能优化

每个版本都带来新功能，Android越来越强大。

厂商的狂欢
#

Android开源免费，任何厂商都可以使用。这导致了Android手机的爆发：

三星：全球最大的Android手机厂商，Galaxy系列是旗舰代表。

小米：性价比策略，在中国和印度市场成功。

华为：曾经是中国最大的手机厂商，后来因制裁转向鸿蒙系统。

OPPO、vivo：线下渠道强大，自拍和快充是卖点。

一加、Realme、荣耀：新兴品牌，各有定位。

Android手机覆盖所有价位：从几百元的入门机到上万元的折叠屏旗舰。智能手机从奢侈品变成大众消费品。

Android的市场份额
#

Android迅速占领市场。

2010年，Android市场份额约23%。

2012年，Android超过iOS，成为市场份额第一。

到2024年，Android占据约71%的全球市场份额，iOS约28%。

但在不同地区，格局不同：

  • 美国：iOS约57%，Android约42%
  • 欧洲：iOS约32%，Android约67%
  • 中国：Android约75%（iOS份额因华为被制裁而下降）
  • 印度：Android约95%

Android在发展中国家占绝对优势，因为价格低廉。

Android的挑战
#

Android也面临挑战：

碎片化：Android有数千种设备，屏幕大小、硬件配置各不相同。开发者需要适配各种设备，测试成本高。

版本分散：很多设备运行旧版本Android，无法获得新功能和安全更新。Google Play Services部分解决了这个问题。

安全：Android允许安装第三方应用，恶意软件更容易传播。

Google控制：虽然Android开源，但Google服务（GMS）是闭源的。没有GMS，Android体验大打折扣。华为被禁止使用GMS后，海外市场暴跌。

安迪·鲁宾的离开
#

2013年，安迪·鲁宾离开Android团队，转向其他项目。

后来，他被指控性骚扰，离开Google。Android由桑达尔·皮查伊接管。

鲁宾创办了Essential公司，发布了一款Android手机，但失败了。公司于2018年关闭。

Android的创始人，没能继续他的传奇。

Android的意义
#

Android的意义是什么？

让智能手机普及：Android让智能手机从奢侈品变成大众消费品。今天，全球有超过30亿人使用Android手机。

开放的力量：Android证明了开源可以成功。一个开源系统，可以与闭源的iOS竞争。

生态的力量：Android生态有数百万应用，覆盖各种需求。

选择的权利：用户可以选择不同品牌、不同价位的Android手机，而不是被一家公司锁定。

下一步：大数据时代
#

移动互联网让数十亿人在线，产生了海量数据。

这些数据被称为"新石油"。谁能掌握数据，谁就能掌握未来。

明天，我们来讲大数据时代。

今日知识点
#

Android：Google开发的移动操作系统，基于Linux内核。Android开源免费，任何厂商都可以使用和修改。Android占据约71%的全球智能手机市场份额。

开源操作系统：源代码公开的操作系统，任何人都可以查看、修改、分发。Android是开源操作系统，iOS是闭源操作系统。开源降低了厂商的成本，但也带来了碎片化问题。

碎片化（Fragmentation）：Android有数千种设备，配置各异。开发者需要适配各种屏幕大小、硬件配置、系统版本，增加了开发成本。

思考题
#

  1. Android开源免费，让智能手机普及到每个人。你觉得开源模式对消费者是好事吗？
  2. Android占据71%市场份额，但iOS占据高端市场的大部分利润。你觉得为什么？
